# **App Name**: Swadeshi Parakh

## Core Features:

- QR Code Scanner: Opens the device camera to scan QR codes.
- QR Code Generator: Generates QR codes based on user input for product name, details, price, and origin.
- Product Origin Validator: Checks the product origin against the Firebase database (swadeshiparakh-default-rtdb.firebaseio.com).
- Origin Indicator: Displays a green symbol if the product is Indian, and a red symbol if it is not, based on the database result.

## Style Guidelines:

- Primary color: Soft saffron (#FF9933) to reflect Indian heritage.
- Background color: Off-white (#F8F8F8) for clean and accessible layout.
- Accent color: Forest green (#228B22) to symbolize the 'Indian' positive indicator.
- Body and headline font: 'PT Sans' for a modern and readable style.
- Use green checkmark and red cross icons for product origin indicators.
- Clean and straightforward layout with clear instructions for scanning and generating QR codes.
- Smooth transitions and animations when displaying scan results.
